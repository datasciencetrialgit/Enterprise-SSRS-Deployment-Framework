# DataSets

Place your SSRS shared dataset files (.rsd) in this directory.

## Shared Datasets

Shared datasets allow you to:
- Reuse queries across multiple reports
- Centralize data logic
- Improve performance through caching
- Maintain consistency across reports

### File naming convention:
- Use descriptive names: `CustomerList.rsd`, `SalesData.rsd`
- Group related datasets with prefixes: `Sales_Monthly.rsd`, `Sales_Yearly.rsd`
- Avoid spaces and special characters
