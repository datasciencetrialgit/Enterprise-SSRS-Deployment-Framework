# Data Sources

Place your SSRS data source files (.rds) in this directory.

## Example Data Sources

You can create data source files using SQL Server Data Tools (SSDT) or by exporting them from an existing SSRS installation.

### Typical data sources include:
- Database connections (SQL Server, Oracle, etc.)
- Web service data sources
- XML data sources
- SharePoint lists

### File naming convention:
- Use descriptive names: `MainDatabase.rds`, `WarehouseDB.rds`
- Avoid spaces and special characters
- Use consistent naming across environments
